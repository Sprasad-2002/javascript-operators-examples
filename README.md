# JavaScript Operators Examples

This repository contains basic examples of different JavaScript operators.

## Operators Covered

- Arithmetic Operators
- Boolean Values
- String Concatenation
- Increment & Decrement Operators
- Logical Operators (AND, OR, NOT)
- Relational Operators
- Type Coercion
- Type Conversion
- Prompt Method Examples

## Folder Structure

Each HTML file demonstrates a specific operator concept in JavaScript.

## Technologies Used

- HTML
- JavaScript

## Purpose

This project is created for learning and practicing JavaScript operator concepts.
